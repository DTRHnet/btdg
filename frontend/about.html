About BTDigg

What is BTDigg?

BTDigg is a BitTorrent DHT search engine that analyzes the DHT network in real-time and provides full-text search over active torrents. This is a clone/demo version created for educational purposes.

Features

  • Real-time DHT Analysis: Continuously crawls the BitTorrent DHT network
  • Full-text Search: Search through torrent titles and descriptions
  • Magnet Links: Direct magnet link generation for easy downloading
  • Recent Findings: View the latest torrents discovered
  • RSS Feed: Subscribe to updates via RSS
  • OpenSearch Integration: Add to your browser's search engines
  • Multi-language Support: Available in multiple languages

How It Works

BTDigg operates by crawling the BitTorrent Distributed Hash Table (DHT) network, which is a decentralized network used by BitTorrent clients to find peers. The system continuously monitors this network, indexes torrent metadata, and makes it searchable through a web interface.

Technology Stack

  • Backend: Python Flask web framework
  • Database: SQLite with full-text search capabilities
  • Frontend: Vanilla HTML, CSS, and JavaScript
  • Search: SQLite FTS5 for fast text search
  • Deployment: Docker containerization
